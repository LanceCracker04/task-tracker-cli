# Task Tracker CLI

A lightweight, dependency-free Command Line Interface (CLI) application built with Python to track your daily tasks and managing a local JSON-based todo list.

## Features
* **Add** tasks with a custom description
* **List** all current tasks showing their status and ID
* **Update** task descriptions dynamically
* **Delete** tasks instantly by ID
* **Mark** tasks as `in-progress` or `done`

## How to Run

### Prerequisite
Make sure you have Python installed on your machine.
